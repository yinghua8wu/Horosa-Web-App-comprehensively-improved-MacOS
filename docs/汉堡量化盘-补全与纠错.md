## 二、调研基线(权威算法与数据)

### 2.1 8 颗 TNP 计算(Swiss Ephemeris,本地 `seorbel.txt` 已逐字节核对)
- Body ID **40–47**:`CUPIDO=40, HADES=41, ZEUS=42, KRONOS=43, APOLLON=44, ADMETOS=45, VULKANUS=46, POSEIDON=47`;`ipl = SE_FICT_OFFSET_1(=39) + 行号`。
- 作为**无摄动开普勒虚拟体**从 `seorbel.txt` 轨道根数算,**历元=分点=J1900**,采用 **James Neely 修订根数**(与 Witte / Sieggrün 原表更吻合)。
- **内禀精度 ~1°(20 世纪内)** —— SE 自身输出的极限,任何 TNP 对拍不要求优于 ~1°。
- 本地 `flatlib-ctrad2/flatlib/ephem/swe.py:155-162` 已映射 40–47,`flatlib/resources/swefiles/seorbel.txt` 存在且数值与权威一致 → 我们的 TNP 是权威算法,**正确**(非 kinastro 线性近似;那是 vendored Streamlit,不进 dial)。

### 2.2 盘面数学
- 90° 盘 = 4 次谐波:`dial = lon mod 90`,四基本点折叠到 0;硬相位 0/90/180/270 重合为「合」,8 次谐波 45° 在 45 标记呈「冲」。45° 盘 = 8 次谐波,22.5° = 16 次谐波。
- 中点 = 半和 `(A+B)/2`,每对有两个相隔 180° 的中点,取近中点(短弧)。

### 2.3 行星图与映点公式
- 行星图(Planetenbilder)敏感点和差式 **`A + B − C = D`**;`X = A/B` 表示 X 落在 A、B 的中点轴上。
- 映点(Spiegelpunkte):
  - **回照 antiscion = `(180 − lon) mod 360`**(跨日至轴 0°巨蟹/0°摩羯镜像)
  - **对映 contra-antiscion = `(360 − lon) mod 360`**(跨分至轴 0°白羊/0°天秤镜像)
  - **注**:90°/45°/22.5° 盘上回照与对映相差 180°≡0(折叠重合),故盘上为**单一「映点」概念**(只在 360° 全圈才分 anti/contra)。

### 2.4 容许度与字形
- Witte 严格 orb ~1°、实用 ~1.5°;Ebertin 宇宙生物学强调 8 次谐波(45°)。
- **TNP 无标准 Unicode 码位**(U+2BE0+ 提案经调研证伪、并未编码)→ 真字形须 SVG / 私用区字体;本轮**保留二字母缩写**(Cu/Ha/Ze/Kr/Ap/Ad/Vu/Po),更清晰且不依赖未编码码位。

## 四、实现明细

### 4.1 共享算法 `utils/uranianDial.js`(纯函数,可单测)
- `antiscion(lon) = (180 − lon) mod 360`;`contraAntiscion(lon) = (360 − lon) mod 360`。
- `planetaryPictures(points, base, orb, opts)`:以个人点/TNP 为锚点(A|B 至少一锚),遍历 C、D;`(A+B−C)` 落 D 的 orb 内即一图。去重键 `sorted(A,B)|C|D`;排序「含个人点 > 含 TNP > 其他」再按 orb;`limit` 默认 40(防 O(n⁴) 过载)。
- `midpointList(points, base, opts)`:无序对各算近中点,同优先级排序。
- `spiegelContacts(points, base, orb, opts)`:`dialSeparation(antiscion(A), B, base) ≤ orb` 即一对映点接触(盘折叠语义,anti/contra 合一)。

### 4.2 后端中点 `astropy/astrostudy/germany/midpoint.py`
- `MidPoint(perchart, orb=1.0, uranian=False)`:
  - **`uranian=False`(默认)**:与历史实现**逐字节一致**,供合盘 `modern/chartcomp.py` 复用,绝不改其输出。
  - **`uranian=True`**(`/germany/midpoint` 专用):
    - 中点对来源 = 行星 + 南北交 + **Asc/MC** + **8 TNP**(`flatlib_ephem.getObject` 走 Swiss Ephemeris)。
    - `_getMidpointsUranian`:无序对(`j>i`)各算一个近中点,消除原 A-B/B-A 双算 + 浮点精确去重的近/远重复。
    - `_getAspectsUranian`:`d = abs(mid.lon − obj.lon) % 360; if d>180: d=360−d`,再判 0/90/180(270 经归一并入 90)——修原 abs 差未归一致的跨 0° 漏判;orb 取构造参数。
    - TNP 也作相位目标。
- `websrv/webgermanysrv.py`:`midpoint()` 解析并透传 `orb`、`MidPoint(..., uranian=True)`;`_build_uranian_tnp` 返 `(out, errors)`,`midpoint()` 把失败写入响应 `tnpError`。
