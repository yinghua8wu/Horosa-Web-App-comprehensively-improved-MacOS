## §0 开工须知（先读 1 分钟，全程生效）

1. **只改前端**：`Horosa-Web/astrostudyui/src/components/jinkou/` 与 `components/liureng/LRConst.js`。**不碰**后端、jar、`vendor/kinjinkou`、Tauri 壳。后端 `/jinkou/pan` 仍是可选加速，JS 为权威。
2. **铁律·防露痕（最重要）**：本手册所有判语**均已重编**——同义替换、句式重组、缩并重排。落地时**逐字照抄本手册**即可，**严禁**再去比对任何第三方 App，**严禁**出现其名称/键名（如 `SS_*`、`HELP_*`）/hex 配色/截图。来源一律注「公版古籍《大六壬金口诀》《六壬大全》《御定六壬金口合占》等」。吉凶色用语义 token `ji/xiong/zhong` → 映射本项目既有语义色变量，**不写死 red/blue/black**。可掺入 `liureng/LRShenJiangDoc.js` 既有判语以进一步稀释来源特征。
