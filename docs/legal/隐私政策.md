# 星阙 Horosa — 隐私政策

**适用产品：** 星阙 Horosa macOS 桌面版（Apple Silicon，macOS 12 及以上）
**最后更新：** 2026-07-02

> **一分钟通俗摘要（非正式，仅帮助理解）**
> - 你的**命盘、案例、偏好、AI 密钥都只存在你自己的电脑上**，我们看不到，也不会上传。
> - 本软件**没有任何遥测、埋点、广告或行为追踪 SDK**。
> - 核心排盘**完全离线可用**。只有三件事会联网，且都由你触发或可控：**检查更新（你手动点击时，GitHub）**、**AI 分析（发给你自己配置的模型）**、**地图选经纬度（高德地图，首次使用前需你在应用内明确同意）**。
> - 使用 AI 分析时，你的命盘文字会发送给你所选择的第三方模型——请知悉并谨慎选择。

## 2. 我们的隐私原则

- **本地优先：** 你的核心数据默认只保存在你的设备上。
- **数据最小化：** 只为实现功能而处理必要的数据。
- **零追踪：** 不使用任何第三方分析、遥测、埋点或广告追踪技术。
- **透明可控：** 联网行为清晰列明；可选功能由你主动开启。

---

## 3. 我们**不**收集、不上传的内容

- 我们**不**在服务器上为使用桌面版核心功能而收集你的**账号信息**——**桌面版无需注册或登录**即可使用本地排盘等核心功能。
- 我们**不**收集你的使用行为、点击、停留时长等**遥测/统计数据**。
- 我们**不**内置任何**广告、第三方分析或追踪 SDK**。
- 我们**不**主动上传你的命盘、案例或分析结果到本项目控制的任何服务器。

> 如本软件将来提供**可选的**在线账号/云同步等功能，其数据处理将在启用前另行、单独告知并征得你的同意，本政策届时相应更新。

## 4. 存储在你设备本地的数据

以下数据保存在你设备的本地应用存储（如浏览器式本地存储 `localStorage`、应用支持目录、本地日志文件）中，**归你所有、由你控制**：

| 数据类别 | 示例 | 说明 |
|---|---|---|
| 命盘 / 排盘资料 | 姓名或代称、出生日期、出生时间、出生地点、经纬度、性别 | 用于排盘计算；**可能包含敏感个人信息**（见第 7 条） |
| 占测 / 事盘案例 | 六爻、奇门、六壬等起卦/起局记录 | 你保存的占测记录 |
| 偏好与设置 | 界面主题、盘式、算法选项等 | 记住你的使用习惯 |
| AI 服务配置 | 你填写的第三方模型服务地址、模型、**接口密钥** | 用于 AI 分析；仅存本地，见第 6.2 条 |
| 本地日志 | 启动与运行诊断日志 | 仅写入本地文件，用于自查问题，**不自动上报** |

**这些数据存放在你的设备上，我们无法访问。** 你可随时在软件内删除相应记录，或通过卸载软件、清除应用数据的方式移除（见第 9 条）。

## 5. 联网功能与出站数据（据实清单）

本软件在**桌面版**下仅在以下三种情形联网。除此之外，代码中出现的其他历史遗留域名在桌面版**不会被调用**。

### 5.1 软件更新检查（你手动触发，可控）
- **目的：** 检查是否有新版本并（经你确认后）下载安装包。
- **接收方：** GitHub（`github.com` / `api.github.com`），由 GitHub 运营。
- **传输内容：** 版本查询请求。此过程会向 GitHub 暴露你的 **IP 地址**及常规网络请求信息（这是任何网络请求的固有特征）。**我们不会因此获得你的身份或命盘数据。**
- **第三方政策：** 受 GitHub 隐私政策约束。
- **可控性：** 检查由你在菜单中手动点击「检查更新」发起，本软件**不做后台定时自动检查**；更新的下载与安装以你的确认为前提。（首次安装、修复或更新本机组件时也会连接 GitHub 下载相应组件。）

### 5.2 AI 分析（可选，你主动触发并自备密钥）
- **目的：** 将你选择的命盘/占测内容发送给大语言模型以生成分析。
- **接收方：** **你自行配置的第三方模型服务**——可能是相关模型厂商的官方接口（其服务器可能位于**中国境外**），或你自建/本地部署的模型。
- **传输内容：** 你所选技法的**命盘/占测快照文本**（可能包含出生日期、时间、地点等信息）以及你的分析指令；使用**你自己的接口密钥**进行鉴权。
- **重要提示：** 一旦启用该功能，相关数据将**离开你的设备**、发送给你选择的第三方，并**受该第三方的条款与隐私政策约束**。请审慎选择服务商、谨慎处理敏感信息。
- **跨境传输：** 若你选择的模型服务器位于你所在国家/地区境外，则相应数据将发生**跨境传输**（见第 8 条）。

### 5.3 地图与经纬度选择（可选，你打开时才加载）
- **目的：** 帮助你在选择出生地/所在地时检索地点并获取经纬度。
- **接收方：** **高德地图 / 阿里巴巴**（内置地图 API）。
- **传输内容：** 地图交互与地点检索请求（如你输入的地名、地图浏览请求），以及作为网络请求固有特征的你的 IP 地址。本软件不向其发送你的命盘等其他数据。
- **第三方政策：** 受高德地图 / 阿里巴巴的服务条款与隐私政策约束。
- **可控性：** 在线地图**默认不加载**——首次使用前，应用会先行提示（将向高德发送你的 IP 与地图请求、其服务器位于中国境内），经你**明确同意**后方才加载；该同意仅记录在本机，可通过清除本软件的本地数据撤回。你也可以使用内置城市快搜（本地数据）或手动直接输入经纬度，完全不触发该第三方请求。

> **本软件不会将上述之外的数据发送到本项目控制的服务器；本项目也不运营用于收集用户数据的后台。**

## 6. 特殊说明：AI 接口密钥与敏感数据

6.1 **接口密钥仅存本地。** 你为 AI 分析填写的第三方接口密钥保存在你设备的本地存储中，用于直接向你所选的模型服务发起请求。**请勿在公共或共享设备上保存密钥**，并注意妥善保管（见《安全说明》）。

6.2 **命盘属敏感信息。** 出生日期、时间、地点等可能构成**敏感个人信息**。当你使用 AI 分析时，这些信息会随快照文本发送给第三方模型。**是否发送、发送给谁，完全由你决定。**

## 7. 儿童与未成年人隐私

本软件含占卜类内容，面向成年人设计。我们不有意收集未成年人的个人信息。未成年人应在监护人知情、同意并指导下使用。若监护人发现未成年人未经同意使用了本软件的联网功能，请通过第 1 条的联系方式与我们联系。

---

## 8. 跨境数据传输

本软件面向全球分发。**本地数据不发生跨境传输。** 仅在你**主动使用**联网功能时可能发生跨境传输：
- **AI 分析**：若你选择的模型服务器位于你所在国家/地区境外，你输入的分析数据将传输至境外接收方；
- **更新检查 / 地图**：相应请求发往 GitHub / 高德的服务器。

上述传输由**你主动触发**，并**受相应第三方的合规机制约束**。在需要征得单独同意的法域，你启用该等功能即视为你知悉并同意相应的跨境传输。若你不希望发生跨境传输，请勿启用相关可选功能，或选择本地/境内的替代方案（如本地部署模型）。

## 9. 你的权利与行使方式

由于你的数据主要**存储在你自己的设备本地**，你可以**直接、即时地**行使对数据的控制：

- **访问 / 查看：** 在软件内查看你保存的命盘、案例与设置。
- **更正：** 在软件内编辑相应记录。
- **删除：** 在软件内删除记录；或卸载软件、清除应用本地数据以彻底移除。
- **可携：** 你可通过软件的导出功能获取你的数据。
- **撤回同意：** 停止使用相关可选联网功能（如不再使用 AI 分析），即撤回相应处理。

依你所在法域的法律，你可能还享有以下权利，我们将在适用范围内予以尊重与配合：

- **中国《个人信息保护法》（PIPL）：** 知情、决定、查阅、复制、更正、删除、注销、要求解释处理规则，以及针对自动化决策的相关权利等。
- **欧盟《通用数据保护条例》（GDPR）：** 访问、更正、删除（被遗忘）、限制处理、数据可携、反对处理，以及向监管机构投诉的权利等。
- **美国加州（CCPA/CPRA）：** 知情、删除、更正、"不出售/不分享个人信息"等权利——**说明：本项目不会出售或分享你的个人信息**。

如需就本地无法自行完成的事项请求协助，或提出投诉，请通过第 1 条的联系方式联系我们。我们将在适用法律要求的期限内回应。你也有权向你所在地的数据保护监管机构投诉。
