# 河洛理数 · 取化工法（左栏选项）+ 运盘节气精度 + 命运篇判断补全（v2.3.0）

> 纯前端（`astrostudyui`）。**不重编 `astrostudyboot.jar`、不重打 runtime。** 分支 `feature/heluo-school-interp`。
> 本版依《河洛理数》(陈抟·邵康节) 古籍原文逐条审计后补全，主链算法零改动。

## 背景

对照《河洛理数》古籍原文（起命篇 / 命运篇）审计现有 heluo 实现，结论：

- **起命 / 起运 / 命运篇判断主链与原文逐条吻合（零算法错误）**：天干取数、地支取数、天地数去 25/30 归卦、五数寄宫（三元）、八卦相荡、起元堂（含乾坤纯卦、三至尊「变而不易」）、先天换后天、得时卦月表（64 卦）、元气/反元气、化工/反化工、纳甲得势——逐条核对均合。
- 差距集中于三类：① 缺**多流派可选项**；② 运盘段界无**节气交时**显示；③ 命运篇有若干「**可算却未显示**」的判断（二十四数名、众宗、顺/反数、正反对体凶、命格综合）。

本版补齐这三类。

**铁律（已落实）**：只服务单一技法的算法选项 → 进该技法**左栏**（镜像参评数「古法/明法」`KinAstroMain` 组件本地 state，不入 globalSetup、不广播事件）；**全局设置 Modal 只放跨技法全局开关**（日界点、晚子时）。

## 一、取化工法 —— 左栏 per-技法选项

- **算法**：`heluoLocal.solarTermHuagong(prevName, tuyong, opts)`。`opts.quHuaGong==='siFangBoOnly'`（直取四方伯）→ 关掉土用补卦，化工仅取四方伯；默认 `'tuWangKunGen'`（土王寄坤艮）→ 四立土用期补 坤艮/反乾兑（＝现状，逐字节不变）。
- **UI（左栏，镜像 canping）**：`KinAstroMain.js` —— state `heluoQuHuaGong: 'tuWangKunGen'`（L764）；左栏「取化工法」`<Select>` 土王寄坤艮/直取四方伯（L1127–1136，紧邻性别）；prop 传入 `quHuaGong={this.state.heluoQuHuaGong}`（center L2082、aux L2250）。
- **消费**：`HeLuoMain.solarTerm()` → `solarTermHuagong(prevName, tuyong, { quHuaGong: this.props.quHuaGong || 'tuWangKunGen' })`（L79）；下游 `judge(...)` 经 `hgOverride` 自动接收，零改。
- **原文依据**：化工气表「春分后震 / 夏至后离 / 秋分后兑 / 冬至后坎 / 四立前后土用坤艮」与 `QUARTER_HG`+`JIEQI_QUARTER` 逐条合。

## 六、典籍口径（已定稿）

1. **土用期 = 四立前 18 日**（采用·现状不变）。此即标准「土王用事」（土旺四季末 18 日）的通行历法定义；古籍「化工气」注「前后各九日」属该传本特定表述。本版按标准土王用事取「四立前 18 日」，`HeLuoMain.solarTerm` 现状即此，**不改动**。
2. **天数 39 归数 = 4**（采用·现状不变）。按「遇十用零数」规则 39−25=14→4 正确；古籍该例「用五数」为笔误。
3. **A1 寄爻顺起：不做（已移除）**。原文只载「寄爻初起（重数+寄爻皆从最下起）」为正法；顺起候选缺典籍出处与算例，本版已**撤除**，`yuanTang` 回归原始单一实现（寄爻初起），不带 `opts`。
