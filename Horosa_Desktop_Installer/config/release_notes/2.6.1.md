> v2.6.1 — on top of v2.6.0, a thorough **AI-mount** polish, **multi-period output** for Zi Wei / Bazi, a new **Bagua dwelling (Feng Shui)** method, and a batch of cross-module fixes. Every technique's chart computation is identical to v2.6.0. Users on older builds receive v2.6.1 automatically on next launch. **The runtime components are updated this release (`2.6.1-runtime1`) and download automatically on upgrade.**

- **AI mount — every per-technique option, nothing missing**: a carpet audit of all mountable techniques — any option a technique's main page exposes (and that changes the mounted snapshot) is also selectable in the gear drawer, so you mount exactly the information you want.
