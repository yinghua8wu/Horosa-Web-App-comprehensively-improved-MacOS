# English

> v3.0.0 — **new "Xuanxue History" module + broad multi-technique school/variant coverage + charting and AI-report enhancements + across-the-board performance work + a batch of bug fixes.** Where schools disagree, the differences are exposed as options; the default chart path is byte-for-byte unchanged. Users on an earlier build receive the v3.0.0 update prompt on launch. **The runtime component is updated this release (`3.0.0-runtime1`) and downloads automatically on upgrade.**

## New module · Xuanxue History
- A standalone "Xuanxue History" page in the tools group, compiled from public-domain classics (the Twenty-Four Histories, Taiping Guangji, etc.): figures, stories, technique lineages, and celestial-event records across the dynasties.
- Offline historical map of China + force-directed figure-relationship graph + chronological timeline, searchable by dynasty / technique / figure.
- Linked to charting: a selected historical moment can be cast directly; original classical texts support cross-reference lookup.

## Multi-technique school coverage
- Fate set: Zi Wei Dou Shu, BaZi, Seven-Governors / Guo Lao, Western astrology (with classical parameters and dignities), and Vedic astrology gain per-school options and comparison views.
- Divination set: Da Liu Ren, Qi Men Dun Jia, Tai Yi, Liu Yao (Na Jia), Jin Kou Jue, Tarot, Feng Shui (six Li-Qi schools), and astronomical geomancy gain school presets and granular variant choices.
- All school disagreements are surfaced as left-panel options with defaults that match prior results; the added derivation layers do not change the byte-level output of the default chart.

## Charting
- Progression charts consolidated: secondary / tertiary / minor progressions unified with the declination and fixed-star views.
- Full parameter propagation: auxiliary-chart and term-system parameters now flow end to end (frontend → backend → compute engine).
- Vedic completeness: more chart styles, divisional sets, and multi-level dasha display.
- Planetarium enhancements: complete Chinese asterisms, precise rise / set / culmination times, lunar-mansion degree grid, and real-time sync during continuous time playback.

## AI analysis
- Per-school snapshot, mounting, restore, and export are aligned across all four surfaces, with content drawn from the actual chart.
- Steadier reasoning tiers and streaming.

## Performance
- Chart cache is persistable and keyed on the runtime version, so upgrading the runtime invalidates stale charts automatically.
- Render guards across charts and removal of redundant recomputation make switching and redraw smoother, with near-instant response across every technique option.

## Stability
- Exhaustive stress matrices over every technique option.
- Numerous interaction fixes across the divination and fate sets plus dark-mode coverage.

The runtime is bumped to `3.0.0-runtime1`.

## Runtime 4 增量修复
- 天文馆：月亮 / 行星位置按真实黄经黄纬速度每帧本地外推，播放丝滑、右栏「黄道坐标 / 星盘位置 / 占星星座」数据实时跟随；暂停瞬间整个场景完全锁定，无任何跳变；点选星体后相机持续跟随（暗星亦支持），手动拖动取消跟随。
- 小工具：八字反查 / 八字格局 / 地平坐标 在后端不可达时静默退出，不再崩页；登录类提示从英文 raw code 改为中文「此操作需要登录后才能使用」。

## Runtime 4 incremental fixes
- Planetarium: Moon / planet positions are now extrapolated locally each frame from the true longitude / latitude speeds, so playback is fluid and the right-rail readings (ecliptic coordinates, chart position, zodiac sign) update in real time; pausing completely locks the entire scene with zero snap-back; after selecting an object the camera tracks it continuously (faint stars included), and any manual drag releases the lock.
- Tools: BaZi reverse-lookup / BaZi pattern / horizontal coordinates silently exit when the backend is unreachable instead of crashing; login-required errors switch from a raw English code to a friendly Chinese hint.
