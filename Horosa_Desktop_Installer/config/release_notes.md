## 安装包怎么选 / Which Package Should I Download?

- 离线安装包：`Horosa-Installer-macos-arm64-offline-pkg.zip`
  - 适合：所有普通用户，尤其是中国大陆、弱网、离线转发、想一次装完的用户
  - 特点：安装包更大，但本机组件已内置，安装完成后可直接打开使用
- 轻量在线 `.pkg` 与 `DMG` 公开入口已取消，不再继续分发，避免首启准备链路和多入口带来的额外不确定性。

- 安装、修复、更新现在统一先进入“安装审查”，列出 app、本机组件和缓存，再由你决定这次要替换哪些内容。
- 离线路径已收紧为“安装完成后直接可用”：如果离线安装结果损坏，启动页会优先引导你重新安装离线包，不再偷偷转为联网修复。
- 启动页、错误态、空状态继续做了一轮 Mac 化收口：主视图优先展示状态与恢复动作，日志和原始错误退到第二层。
- 修复了主窗口默认尺寸异常的问题；现在启动后会默认最大化铺满屏幕，但不会进入独立全屏空间。
- 修复了节气盘左上角黄道/分宫制偶发显示 `undefined` 的问题，现在会稳定显示当前黄道与分宫制。
- 修复了节气盘 `XX星盘` 不跟随上方黄道与分宫制选择重算的问题；切换后会立即按当前设置重新计算，而不是卡在“回归黄道 + 整宫制”。
- 本次发布版本号为 `v1.1.2`，并同步发布独立 runtime `1.1.2-runtime1`，确保离线安装、应用更新与这次节气盘修复都落在同一套运行时链路上。
