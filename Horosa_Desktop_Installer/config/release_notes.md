## 安装包怎么选 / Which Package Should I Download?

- 离线安装包：`Horosa-Installer-macos-arm64-offline-pkg.zip`
  - 适合：所有普通用户，尤其是中国大陆、弱网、离线转发、想一次装完的用户
  - 特点：安装包更大，但本机组件已内置，安装完成后可直接打开使用
- DMG 补充入口：`Horosa-Desktop-macos-arm64.dmg`
  - 适合：偏好标准拖拽安装路径、并愿意在 app 内完成首次准备的 Mac 用户
  - 特点：拖入 `Applications` 后直接打开；如缺少本机组件，会在 app 内完成首次准备
- 轻量在线 pkg 已取消，不再继续分发，避免首启联网准备带来的额外不确定性。

- 安装、修复、更新现在统一先进入“安装审查”，列出 app、本机组件和缓存，再由你决定这次要替换哪些内容。
- 离线路径已收紧为“安装完成后直接可用”：如果离线安装结果损坏，启动页会优先引导你重新安装离线包，不再偷偷转为联网修复。
- 启动页、错误态、空状态做了一轮 Mac 化收口：主视图优先展示状态与恢复动作，日志和原始错误退到第二层。
- 修复了“推运盘 > 主/界限法”首次进入时表格不显示、点击计算没有立即刷新的问题；现在首次进入主/界限法就会直接拉取并显示主限法表格，不需要先切到主限法盘。
- 新增独立的偏好设置与诊断中心窗口，窗口管理、菜单和恢复入口更接近正式 Mac app。
- 更新流程更稳：应用内更新会先做安装审查，再下载、校验、替换并自动重开，同时保留用户数据。
- Tooltip、滚动和窗口细节继续收口，避免长内容贴边被截断，整体体验更像桌面应用而不是工程安装器。
- 本次发布版本号为 `v1.1.1`，并同步发布独立 runtime `1.1.1-runtime1`，确保 DMG 首次准备、离线安装与应用内更新都落在这次修复后的同一套运行时链路上。
